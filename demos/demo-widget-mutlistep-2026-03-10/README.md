# Widget Drag-Drop Demo

A drag-and-drop dashboard grid demo built with React, TypeScript, and [@dnd-kit](https://dndkit.com/).

## Prerequisites

- Node.js (v18+)
- npm

## Install

```sh
npm install
```

## Run (dev server)

```sh
npm run dev
```

Opens at `http://localhost:5173` by default.

Alternatively, without installing:

```sh
npx vite
```

## Build

```sh
npm run build
```

Output is written to `dist/`.

## Test

```sh
npm test
```

Runs the [Vitest](https://vitest.dev/) test suite.
